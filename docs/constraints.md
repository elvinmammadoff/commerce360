# Constraints

This is NOT a template.

This is NOT a landing page clone.

This is NOT an AI toy.

Everything should feel like a real company.

No placeholder Lorem Ipsum.

Use realistic company names.

Use realistic product names.

Use realistic analytics.

Use realistic invoices.

Use realistic pricing.

Use realistic credits.

Use realistic jobs.

Avoid fake-looking UI.

Every page must be believable enough that a customer would think the product is already live.

Every interaction should reinforce trust.

Design for long-term scalability.

Future backend integration must require minimal frontend changes.
